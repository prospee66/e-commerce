üîß TROUBLESHOOTING
===================

Navigation links not working?
‚Üí Make sure app is running (see "Quick Start" above)
‚Üí Check URL is http://localhost:3000
‚Üí Press F12 and check Console for errors
‚Üí Try Ctrl + F5 to hard refresh

Can't see navigation menu?
‚Üí On desktop: Make browser window wider (>768px)
‚Üí On mobile: Click menu icon (‚ò∞) in top right
‚Üí Try zooming out: Press Ctrl + 0

Links click but nothing happens?
‚Üí Check browser console (F12)
‚Üí Make sure JavaScript is enabled
‚Üí Try different browser
